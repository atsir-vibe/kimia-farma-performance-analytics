# Kimia Farma Performance Analytics

Dashboard analisis kinerja Kimia Farma periode 2020–2023 menggunakan Google BigQuery dan Looker Studio.

## Tujuan
Menganalisis tren penjualan, profit, dan performa cabang untuk mendukung pengambilan keputusan bisnis.

## Tools
- Google BigQuery
- SQL
- Google Looker Studio

## Dataset
Data transaksi Kimia Farma periode 2020–2023 yang mencakup:
- Transaksi penjualan
- Data cabang & provinsi
- Rating cabang & transaksi

## Proses Analisis
- Import dan validasi dataset ke BigQuery
- Pembuatan tabel analisa menggunakan SQL
- Perhitungan KPI (Net Sales, Net Profit, Total Transaksi)
- Visualisasi dashboard interaktif

## Output
- Dashboard Performance Analytics Kimia Farma
- Insight tren penjualan, profit, dan performa cabang
